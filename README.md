# BLR Property Visit Guide

A mobile-first PWA for property visits in Bangalore. Three tabs:
- **Properties** — ranked visit list with addresses, contact approach, dates
- **Questions** — 20 questions filtered by who to ask (Seller / RWA / Resident / Broker)
- **Checklist** — 27-point on-site checklist with tap-to-check and offline persistence

## Deploy to GitHub Pages (one-time setup)

1. Push this repo to GitHub
2. Go to repo **Settings → Pages**
3. Source: **Deploy from a branch** → branch: `main` → folder: `/ (root)`
4. Save — your URL will be `https://jaeck.github.io/drive-empty/`
5. On your phone: open the URL → tap Share → **Add to Home Screen**

The app works fully offline once loaded.
